# Pagination

Previous / Next navigation for paginated data sets. Renders a simple control row with a page indicator between the two buttons.

---

## When to use

- At the bottom of a Table to navigate pages of results.
- Anywhere you split a list into pages and need simple navigation controls.
- For cursor-based APIs you can show only Previous/Next and drive state externally.

## Pattern: Table + Pagination

```tsx
<Stack gap="nano">
  <Card padding="none">
    <Table columns={columns} rows={pageRows} />
  </Card>
  <Inline justify="flex-end">
    <Pagination page={page} totalPages={totalPages} onPageChange={setPage} />
  </Inline>
</Stack>
```
