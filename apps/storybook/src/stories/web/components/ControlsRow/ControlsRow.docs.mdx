# ControlsRow

A toolbar layout primitive. Distributes content between a left group and a right group using `justify-content: space-between`. Items within each group flow in an `Inline` with `nano` gap.

---

## When to use

- Table toolbars — search/filter controls on the left, action buttons on the right.
- Page headers — title/breadcrumbs on the left, primary actions on the right.
- Any layout that needs consistent left-right distribution at full width.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `left` | `React.ReactNode` | — | Content anchored to the left group |
| `right` | `React.ReactNode` | — | Content anchored to the right group |
| `gap` | `CreSpace` | `'micro'` | Outer gap between the two groups (used by the outer Inline) |
| `wrap` | `boolean` | `true` | Whether groups wrap onto a new line on small viewports |
| `className` | `string` | — | Additional CSS class |
| `style` | `React.CSSProperties` | — | Inline styles |

---

## Data page pattern

ControlsRow is designed to sit above a Card + Table:

```tsx
<Stack gap="micro">
  <ControlsRow
    left={
      <>
        <Input value={query} onChange={setQuery} placeholder="Search" />
        <Select value={status} onChange={setStatus} options={statusOptions} />
      </>
    }
    right={
      <>
        <Button onClick={handleReset}>Reset</Button>
        <Button onClick={handleExport}>Export</Button>
      </>
    }
  />
  <Card padding="none">
    <Table ... />
  </Card>
</Stack>
```
