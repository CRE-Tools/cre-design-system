# Generic Data Page

A full-page composition demonstrating how to combine design system primitives and components into a functional data management interface. This example shows a typical table view with search, filtering, pagination, and detail drawers.

---

## Features demonstrated

- **Search and filtering** — Text search, dropdown status filter, and date range filter
- **Field selector** — Dynamically show/hide table columns
- **Pagination** — Navigate through large datasets
- **Detail drawer** — View record details in a side panel
- **Modal** — Confirmation dialogs and focused tasks
- **Empty states** — Graceful handling of no-results scenarios
