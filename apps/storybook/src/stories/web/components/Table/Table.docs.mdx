# Table

A data table with optional column sorting and row selection. Table is purely presentational — all data and state live in the parent. Pair with `Card`, `ControlsRow`, `Pagination`, and `EmptyState` to build complete data pages.

---

## When to use

- Displaying a list of records with multiple attributes per row.
- Sortable columns for comparing values.
- Row selection for bulk operations.
- Do **not** use Table for key/value pairs — use a `Stack` of `Field` components instead.

---

## Basic (with sorting)

Columns with a `sort` function render a clickable header button. Click once to sort ascending, again for descending.

## Full data page pattern

```tsx
<ControlsRow
  left={<><Input value={query} onChange={setQuery} /><Select ... /></>}
  right={<Button>Export</Button>}
/>
<Card padding="none">
  <Table columns={columns} rows={pageRows} emptyState={<EmptyState ... />} />
</Card>
<Pagination page={page} totalPages={totalPages} onPageChange={setPage} />
```
