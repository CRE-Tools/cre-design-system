# Checkbox

A binary toggle input that also supports an indeterminate (mixed) state. Commonly used in forms and table row selection.

---

## When to use

- Boolean settings and opt-in preferences.
- "Select all" controls for a list — pair with the `indeterminate` prop when some but not all items are selected.
- Individual row selection in a Table.
- For a single forced choice use `Select`; for mutually exclusive options use a radio group.

## Notes

- If both `indeterminate` and `checked` are true, `checked` takes precedence and indeterminate is cleared.
- Always provide either `label` or `ariaLabel` — a checkbox without accessible text fails WCAG SC 1.3.1.
