# Button

Buttons are used to trigger user actions across CRE applications.

Common usage patterns:

- Primary actions (e.g. "Confirm", "Continue", "Save")
- Secondary actions (e.g. "Cancel", "Back")
- Inline actions inside panels (e.g. "Edit", "Remove")

All visual values come from CSS custom properties set by `CreThemeProvider`. The button never references raw hex values — it is fully theme-driven and supports light and dark mode out of the box.

## Size variants

Three sizes are available via the `size` prop:

| Size | Use case | Font | Height |
|---|---|---|---|
| `regular` | Standard web UI | 15px / 500 | 40px fixed |
| `large` | Desktop primary actions | 16px / 500 | auto (12px vertical padding) |
| `vr` | VR / spatial UI, extra-large touch targets | 24px / 400 | auto (16px vertical padding) |
