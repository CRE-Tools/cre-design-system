# Button

Buttons are used to trigger user actions across CRE applications.

Common usage patterns:

- Primary actions (ex: “Confirm”, “Continue”, “Save”)
- Secondary actions (ex: “Cancel”, “Back”)
- Inline actions inside panels (ex: “Edit”, “Remove”)

This component is theme-driven by default (Light/Dark) and can also be overridden per instance via the `colors` prop.
