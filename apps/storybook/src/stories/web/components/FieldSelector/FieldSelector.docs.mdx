# FieldSelector

A small “columns” trigger button that opens a popover of checkboxes for toggling which fields/columns are visible.

`FieldSelector` is designed to be controlled: you own the `visibleFields` array and update it via `onVisibleFieldsChange`.

---

## When to use

- Table or list views where users can customize visible columns.
- Dense toolbars where the trigger needs to stay compact.

---

## Field derivation

The list of all selectable fields comes from:

- `fields` (explicit list of dot-paths), or
- `data` (rows) which are flattened into dot-paths via `flattenFields`.

If neither is provided, the popover shows “No fields available.”
