# Select

A custom single-select dropdown with keyboard navigation. Replaces the native `<select>` with a token-styled dropdown that matches the Input appearance. Always pair with `Field` for a label and error state.

---

## When to use

- Choosing one option from a small-to-medium list (2–20 items).
- Filters, status pickers, category selectors.
- For boolean choices use `Checkbox`; for large lists consider search-driven alternatives.

## Keyboard behaviour

| Key | Action |
|-----|--------|
| `Enter` / `Space` | Open the dropdown |
| `ArrowDown` / `ArrowUp` | Navigate options |
| `Enter` | Select highlighted option |
| `Escape` | Close without selecting |

---

## With Field

```tsx
<Field label="Status" htmlFor="status-select">
  <Select
    id="status-select"
    value={status}
    onChange={setStatus}
    placeholder="Pick a status…"
    options={[
      { value: 'active', label: 'Active' },
      { value: 'archived', label: 'Archived' },
    ]}
  />
</Field>
```
