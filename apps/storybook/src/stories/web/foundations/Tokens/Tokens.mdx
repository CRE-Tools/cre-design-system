# Tokens

Tokens are the **single source of truth** for colours and component states in the CRE Web UI.

The system is layered:

- **Palette tokens** — raw colour ramps from Figma (e.g. `primary.600`).
- **Semantic tokens** — design-intent names (e.g. `bg`, `text`, `border`, `accentBg`).
- **Component tokens** — stateful component values (e.g. button `hoverBg`, `disabledFg`).

Theme mode (Light/Dark) is controlled from the Storybook toolbar (`Theme`). See the **Design Tokens** story for a live visual reference of all token values.

## Semantic tokens (Light vs Dark)

{(() => {
  const light = createThemeTokens('light');
  const dark = createThemeTokens('dark');

  const rows = [
    { name: 'bg',             light: light.semantic.bg,             dark: dark.semantic.bg },
    { name: 'surface',        light: light.semantic.surface,        dark: dark.semantic.surface },
    { name: 'surfaceRaised',  light: light.semantic.surfaceRaised,  dark: dark.semantic.surfaceRaised },
    { name: 'text',           light: light.semantic.text,           dark: dark.semantic.text },
    { name: 'textMuted',      light: light.semantic.textMuted,      dark: dark.semantic.textMuted },
    { name: 'textSubtle',     light: light.semantic.textSubtle,     dark: dark.semantic.textSubtle },
    { name: 'border',         light: light.semantic.border,         dark: dark.semantic.border },
    { name: 'borderStrong',   light: light.semantic.borderStrong,   dark: dark.semantic.borderStrong },
    { name: 'focusRing',      light: light.semantic.focusRing,      dark: dark.semantic.focusRing },
  ];

  const cellStyle = { padding: 8, borderBottom: '1px solid var(--cre-color-border)' };

  return (
    <div style={{ overflowX: 'auto' }}>
      <table style={{ borderCollapse: 'collapse', width: '100%' }}>
        <thead>
          <tr>
            <th style={thStyle}>Token</th>
            <th style={thStyle}>Light</th>
            <th style={thStyle}>Dark</th>
          </tr>
        </thead>
        <tbody>
          {rows.map((r) => (
            <tr key={r.name}>
              <td style={cellStyle}><code>{r.name}</code></td>
              <td style={cellStyle}>
                <span style={{ display: 'inline-flex', alignItems: 'center', gap: 8 }}>
                  <span style={{ width: 14, height: 14, borderRadius: 4, background: r.light, border: '1px solid var(--cre-color-border)', flexShrink: 0 }} />
                  <code style={{ fontSize: 11 }}>{r.light}</code>
                </span>
              </td>
              <td style={cellStyle}>
                <span style={{ display: 'inline-flex', alignItems: 'center', gap: 8 }}>
                  <span style={{ width: 14, height: 14, borderRadius: 4, background: r.dark, border: '1px solid var(--cre-color-border)', flexShrink: 0 }} />
                  <code style={{ fontSize: 11 }}>{r.dark}</code>
                </span>
              </td>
            </tr>
          ))}
        </tbody>
      </table>
    </div>
  );
})()}

## CSS variable contract

All theme tokens are exposed as `--cre-*` CSS custom properties by `CreThemeProvider`. Components reference only these variables — never raw token values — so they respond to mode changes automatically.

{(() => {
  const vars = themeTokensToCssVars(createThemeTokens('light'));
  const entries = Object.entries(vars).sort(([a], [b]) => a.localeCompare(b));

  const cellStyle = { padding: 8, borderBottom: '1px solid var(--cre-color-border)' };
  const thStyle = { ...cellStyle, textAlign: 'left', fontWeight: 600 };

  return (
    <div style={{ overflowX: 'auto' }}>
      <table style={{ borderCollapse: 'collapse', width: '100%' }}>
        <thead>
          <tr>
            <th style={thStyle}>CSS variable</th>
            <th style={thStyle}>Value (light mode)</th>
          </tr>
        </thead>
        <tbody>
          {entries.map(([key, value]) => (
            <tr key={key}>
              <td style={cellStyle}><code style={{ fontSize: 11 }}>{key}</code></td>
              <td style={cellStyle}><code style={{ fontSize: 11 }}>{value}</code></td>
            </tr>
          ))}
        </tbody>
      </table>
    </div>
  );
})()}
