# Container

Centers page content horizontally and constrains its maximum width using layout tokens. Always apply horizontal padding so content doesn't touch viewport edges.

---

## When to use

- Wrapping top-level page content.
- Constraining a content column to a readable or structured width.
- Do **not** use Container inside another Container — nest layout primitives (Stack, Grid) inside a single Container.
