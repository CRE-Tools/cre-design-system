# Divider

A semantic separator (`<hr>` or `<div role="separator">`) that uses `--cre-color-border` for consistent colour across themes.

---

## When to use

- Separating sections inside a card, panel, or toolbar.
- Vertical dividers in a horizontal toolbar to group actions.
- Avoid overusing — rely on spacing (Stack gap) as a first tool before adding a visual line.

## Notes

- Divider always renders as `<div role="separator">` with the appropriate `aria-orientation` attribute.
- Vertical Divider requires a parent with an explicit height — it is `height: 100%` by default.
- Colour is always `--cre-color-border` and responds to light/dark mode automatically.
